# Ведически астрологичен калкулатор

Отворен уеб калкулатор на български език за:

- D-1 (Раши) в северен стил
- D-9 (Навамша) в северен стил
- отделна D-1 транзитна карта за сравнение с рождената
- таблица с Асцендент и планети по знак, градуси и накшатра
- автоматично или ръчно задаване на timezone
- Vimshottari Dasha с Mahadasha, Antardasha и Pratyantardasha

## Локално пускане

Проектът е тестван с Python 3.11.

```powershell
python -m pip install -r requirements.txt
python app.py
```

След това отвори `http://127.0.0.1:5000`.

## Бележки

- По подразбиране ayanamsha е `Lahiri`.
- За Rahu/Ketu има избор между среден и истинен възел.
- Часовата зона се определя по координати и дата, но може да бъде коригирана ръчно.
- Vimshottari Dasha използва `mean tropical solar year = 365.24219 дни`.
